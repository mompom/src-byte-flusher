🤖 Byte Flusher

Home · Text Flush · File Flush · Auto Scroll · Docs · GitHub

Send text via BLE (Web Bluetooth) to the Flusher, which types it accurately on the Target PC as USB keyboard (HID) input.

Cautions (Important)

  • Intended for testing/development automation/demo purposes on systems you own/manage or in explicitly authorized environments.
  • Do not use for unauthorized access, security bypass, unauthorized input, or any illegal/unethical purpose.
  • Due to keyboard input automation, input may go to unintended locations due to focus changes/popups/UAC/notifications. Verify in a test environment first.
  • Control PC: Chrome/Edge (Web Bluetooth)
  • Flusher: BLE receive → buffer → USB HID keyboard output
  • Target PC: Receives USB keyboard input to create text/scripts
